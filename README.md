# 📧 Email Template Design – CodSoft UI/UX Internship (Task 2)

## 📌 Overview
This repository contains **Task 2** of my **UI/UX Design Internship at CodSoft** — designing a professional, promotional **Email Template**.

An email template is a pre-designed layout used to send marketing or promotional emails. A good template makes it quick and easy for businesses to send emails that look professional, on-brand, and engaging — without designing from scratch every time.

## 🎯 Task Objective
Design a clean, visually appealing email template suitable for a **newsletter / product update email**, following core UI/UX principles for email design.

## 🖼️ Preview
The design (`Email_template.png`) includes:
- A header with brand logo and "Weekly Newsletter" title
- A hero section announcing big news, with supporting text and a **CTA button** ("Go to your account!")
- A "New Releases" section highlighting product features (Timer, Measure Tool, Slider, Users) with icons and descriptions
- A second CTA button reinforcing the primary action
- A footer with social media icons, subscription note, and an **Unsubscribe** link

## ✅ Key Design Principles Followed

### 1. Visually Appealing
Careful use of color (orange/peach brand palette), consistent typography, and custom icons/illustrations to keep the email engaging and on-brand.

### 2. Easy to Use
Clear visual hierarchy — headline, supporting copy, and a single obvious call-to-action button — so recipients instantly know what to read and what to click.

### 3. Responsive
Layout structured in simple, stacked sections (single column blocks) so it adapts well across desktop, tablet, and mobile screens.

**Font family:** The rounded, geometric sans-serif look (seen in "Weakly Newsletter" and headings) is consistent with fonts like **Poppins, Montserrat, or Inter** — common choices for modern marketing emails.

**Spacing/Layout notes:**
- Generous white space between sections (hero, features, footer)
- Angled/diagonal section dividers (skewed background shape) separate the peach and gray zones for visual interest
- 2-column grid used for the feature icons (Timer/Measure Tool, Slider/Users)
- Centered CTA buttons with rounded corners (~8–10px border-radius) and drop shadow for depth

## 🧩 Step-by-Step: How This Project Was Built (Figma Workflow)

### Step 1: Setup
1. Open Figma and create a new file.
2. Set the artboard/frame width to **1252px** (standard email width) with auto height.
3. Create a color style library using the hex codes above so they're reusable across the design.

### Step 2: Header Section
1. Draw the top background rectangle and apply the peach gradient (`#FFE4CC` → `#FDD5B0`).
2. Place the brand logo (orange diamond with bar-chart icon) in the top-left using the vector/pen tool or an imported icon.
3. Add the text **"Weakly Newsletter"** top-right, bold, ~32px, dark color.

### Step 3: Hero Section
1. Add the heading **"Don't miss out our Big news!"** on the left, bold ~34px.
2. Below it, add a paragraph text block (Lorem Ipsum placeholder), 15–16px, gray, with comfortable line-height.
3. Create the CTA button:
   - Rectangle with 8–10px corner radius
   - Fill with orange gradient
   - Add centered white bold text "Go to your account!"
4. On the right side, place/import the envelope + notification illustration (can use an icon set like unDraw, Storyset, or Freepik illustrations).

### Step 4: Section Divider
1. Draw a skewed/angled rectangle shape that transitions from the peach background to the light gray background — this creates the diagonal-cut effect between sections.

### Step 5: "New Releases" Section
1. Set background to light gray (`#F2F2F2`).
2. Add the small orange eyebrow label **"NEW RELEASES"**, bold, letter-spaced.
3. Add the large heading **"Checkout our amazing features"** below it, bold ~44px.
4. Build a **2x2 grid** of feature blocks:
   - Circle shape (colored per feature: green, yellow, red, blue)
   - Icon centered inside the circle (Timer, Ruler, Slider, Users)
   - Bold title below the icon (~22px)
   - Description paragraph below the title (~14px, gray)
5. Add a second CTA button (same style as before) centered below the grid.

### Step 6: Footer Section
1. Set footer background back to peach, matching the header.
2. Place the logo icon again (smaller, top-right of footer).
3. Add disclaimer text: "You received this email because you signed up on our website or made a purchase from us."
4. Add the **Unsubscribe** link, underlined.
5. Add social media icons (Facebook, Twitter, Instagram) in colored circular badges, bottom-left.

### Step 7: Consistency & Polish
1. Apply consistent padding/margins across all sections (e.g., 80–100px side margins).
2. Use Figma's **Auto Layout** for each section so spacing stays consistent if content changes.
3. Group and name all layers properly (Header, Hero, Features, Footer) for a clean file structure.
4. Check color contrast for accessibility (text vs. background).

### Step 8: Export & Documentation
1. Export the final design as PNG/JPG for preview (`Email_template.png`).
2. If converting to a real, sendable email, rebuild the layout using **table-based HTML + inline CSS** (email clients like Outlook/Gmail don't support modern CSS well), keeping widths responsive with `max-width` and media queries for mobile.
3. Document the process (this README) and upload to GitHub/LinkedIn as part of the internship submission.
